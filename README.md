# HealthApp
Avoir un compte snowflake

créer clé privée et publique
créer l'utilisatuer de service deployment_user
assigner la clé publique à l'utilisateur

executer la commande export SNOWFLAKE_ACCOUNT='[renseigner le account locator]'
executer la commande export SNOWFLAKE_USER='[renseigner le nom d'utilisateur]'
executer la commande export SNOWFLAKE_PASSWORD='[mot de passe de la session snowflake]'
executer le script main_script.sh
